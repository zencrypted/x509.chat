ОГОЛОШЕННЯ

CHAT X.509 анонсує формалізовану модель і специфікацію для мультікаст та юнікаст пабліш-сабскрайб брокерів і їх конфігурацій, а також референсні імплементації для сучасних оптимізованих підприємств на базі телекомунікаційної платформи Ericsson/OTP, які прагнуть стабільності і повноти, покладаючись на стандарти, які повільно міняються і не потребують перегляду десятиліттями.

Специфікація на месаджінговий тракт піднімає питання ораганізації і топології складних, часто ненадійних мереж, а пов'язані з ним стандарти на серіалізатори визначають модель інфраструктури підприємства. Базуючись на зрілому стандарті X.680 наша компанія пропонує ASN.1 компілятори для мов C99, Java, Swift, Go, TypeScript, Rust.

CHAT X.509 організація спеціалізується на розробці безпечних систем миттєвого обміну повідомленнями та ключових компонентів публічної інфраструктури ключів (PKI). Наші проєкти основані на стандартах ASN.1, X.509, CMS/S-MIME, LDAP, DNS та сучасних криптографічних протоколах (включаючи постквантову криптографію та MLS).

CHAT X.509

CHAT X.509 — це безпечна програма обміну повідомленнями з відкритим кодом на основі відкритого ключа, яка використовує сервери лише у якості брокера для маршрутизації і надсилання наскрізних зашифрованих повідомлень без доступу до переписки та метаданих користувачів. Програмне забезпечення засноване на принципах конфіденційності, для нього не потрібен номер телефону чи будь-яка інша особиста інформація.

  • • Протокол v0 — WebSocket (Mail)
  • • Протокол v1 — UDP (Message)
  • • Протокол v2 — TCP (Chat)
  • • Протокол v3 — MQTT (Broker)

CHAT X.509 належить до класу мультипрофільних месенжерів, захищених за допомогою ECC PKI сертифікатів. Чат побудований на телекомунікаційній платформі Erlang/OTP від Ericsson, підтримуює KYBER і MLS, часто змінює ключі та приймає і надсилає повідомлення в S/MIME сумісних CMS конвертах.

Чат реалізований як простий сервер доставки миттєвих повідомлень розроблений для ненадійних мереж та у відповідності до стандартів ISO/IETF. Він використовує монолітний асинхронний TCP сервер, LDAP сервер для корпоративних ієрархічних конфігурацій, сервер імен UNIX та CA сервер для видачі ключів:

  • • LDAP для директорії користувачів;
  • • DNS для безпеки іменного простору;
  • • CA для видачі сертифікатів.
